Buenas Prácticas de Nomenclatura

Clases, Métodos y Comentarios

Clases

Las clases deben nombrarse con PascalCase, iniciando cada palabra con mayúscula. Representan entidades del sistema.

class ClientePreferencial {
// contenido de la clase
}

Métodos

Los métodos realizan acciones y se escriben con camelCase. Inician con un verbo.

void calcularDescuento();
function obtenerSaldoCliente() { // código del método }

Comentarios

Los comentarios ayudan a comprender el código. Sé claro, preciso y útil.

// Verifica si el usuario está autenticado
// Calcula el total del pedido con IVA

Elemento Estilo Ejemplo
