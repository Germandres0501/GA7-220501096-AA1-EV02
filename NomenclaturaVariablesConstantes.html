Nomenclatura de Variables y Constantes

¿Qué son?

Son nombres que usamos para identificar datos en la memoria de un programa. Una buena nomenclatura facilita la lectura, mantenimiento y entendimiento del código.

Reglas generales

  • Deben comenzar con letra (no números ni caracteres especiales)
  • No deben contener espacios ni acentos
  • Usar nombres descriptivos que indiquen su función
  • Evitar abreviaturas excesivas o confusas

Estilos comunes de nomenclatura

Estilo Ejemplo Uso común
camelCase nombreUsuario Variables y funciones en JavaScript, Java
PascalCase NombreUsuario Clases y constructores
snake_case nombre_usuario Python, bases de datos
UPPER_CASE MAX_TAMANO Constantes

Ejemplos prácticos

let nombreUsuario = "Andrés";
const MAX_INTENTOS = 5;
let resultado_final = 98;

Consejos adicionales

  • Usa el inglés si tu código está en proyectos internacionales
  • Mantén la consistencia en todo tu proyecto
  • Documenta el significado si el nombre no es 100% claro

Recuerda: un buen nombre puede ahorrarte horas de confusión en el futuro.
